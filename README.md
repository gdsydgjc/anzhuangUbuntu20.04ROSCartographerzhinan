1. **Ubuntu 20.04安装指南**：详细介绍了如何在计算机上安装Ubuntu 20.04操作系统。

3. **Cartographer安装指南**：详细说明了如何在安装了ROS的Ubuntu 20.04系统上安装和配置Cartographer。
